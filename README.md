##  布局

常用两个：
- LinearLayout(线性布局)

- RelativeLayout(相对布局)

其他：

- TableLayout (表格布局)

- FrameLayout (帧布局)

- GridLayout (网格布局) 安卓4.0后引入

- AbsoluteLayout (绝对布局) 


## 表单控件

- TextView (文本框)

- EditText (输入框)

- Button (按钮)

- ImageView (图像视图)

- RadioButton(单选按钮)&Checkbox(复选框)

- ToggleButton(开关按钮)

- ProgressBar(进度条)

- SeekBar (拖动条)

- RatingBar (星级评分条)

## 控件

- ScrollView (滚动条)

- Date & Time (日期/时间)

- **Adapter** (适配器)
